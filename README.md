# RJC juegos didacticos
> Esta es una app para el emprendimiento de un amigo que se dedica a la fabricacion de juegos didácticos en madera para niños y toda la familia.
La aplicación tiene como objetivo visualizar sus productos, mejorar la experiencia de compra y aumentar las ventas, al mostrar de manera atractiva los productos disponibles y permitir una compra fácil.


La aplicación busca ser un puente entre los juegos didácticos tradicionales, fabricados a mano en medera, y los usuarios interesados en productos educativos de calidad. A través de la app, los usuarios pueden explorar los productos disponibles, conocer sus características y realizar compras cómodamente.
